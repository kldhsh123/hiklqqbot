# 掷骰子

- **作者**: [24xx42](https://github.com/42xx42)
- **插件文件**: `roll_plugin.py`
- **命令**: `/roll`
- **描述**: 按常见骰子表达式生成随机结果，适合桌游、群聊互动或快速决策

## 安装

1. 将 `roll_plugin.py` 复制到 HiklQQBot 主仓库的 `plugins/` 目录
2. 启动机器人，或执行 `/hiklqqbot_reload`

## 使用方法

```text
/roll
/roll d20
/roll 2d6+3
/roll 3d8-1
```

## 表达式格式

- 基本格式：`NdM`
- 可选修正值：`NdM+K` 或 `NdM-K`
- 省略 `N` 时默认为 `1`

例如：

- `d20` 表示投掷 1 个 20 面骰
- `2d6+3` 表示投掷 2 个 6 面骰，再加上 3
- `3d8-1` 表示投掷 3 个 8 面骰，再减去 1

## 默认值与限制

- 不带参数时，默认投掷 `1d100`
- 骰子数量范围：`1-20`
- 骰子面数范围：`2-1000`
- 修正值绝对值不能超过 `100000`

## 示例输出

```text
🎲 2d6+3 = [4, 6] + 3 = 13
```

## 错误提示

参数格式不合法时，插件会返回示例提示：

```text
格式错误。示例：/roll、/roll d20、/roll 2d6+3
```
